This project features a wireless communication system designed to synchronize race start signals while providing haptic feedback for hearing-impaired participants. Built using ESP32 and NRF24L01 modules, the system ensures ultra-low latency transmission, which is critical for race synchronization accuracy.

A web server interface allows users to easily configure and monitor the devices in real time. The communication protocol is implemented using the RadioHead library, and a custom PCB was designed in KiCad to integrate the components efficiently.

Overall, this project demonstrates a fast, accessible, and reliable haptic signaling system optimized for competitive events.
